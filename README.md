# 🍑 Peach

Peach is a modern dating platform built to create smooth interactions, profile discovery, matching, and messaging between users.

This project was developed as a team project. I mainly worked on the backend side of the application using Spring Boot and related technologies.

---

# ✨ Features

## 👤 Authentication & Security
- Secure signup and login flow
- JWT based authentication and authorization
- Password hashing using BCrypt
- Protected APIs with Spring Security
- Custom request filtering and validation

## ❤️ Matching System
- Swipe based matching logic
- Mutual like detection for automatic match creation
- Prevention of duplicate swipes
- Prevention of self swipes
- Transaction safe match handling

## 💬 Messaging System
- One to one chat support between matched users
- Ordered chat history retrieval
- Last message preview handling
- Unread message count tracking
- Authorization checks before accessing chats

## 🧾 Profile Management
- Profile creation and update APIs
- Feed generation logic
- DTO based request and response handling
- Validation for cleaner API responses
